# MO Rubrics

This folder stores rubric files used by the MO proof-evaluation preparation
script.

- `imo25_guideline.md`: IMO 2025 per-problem grading guidelines used
  with `prepare_mo_eval.py --guideline-md`.
- `usamo2026_guideline.md`: USAMO 2026 per-problem grading guidelines used
  with `prepare_mo_eval.py --guideline-md`.

Example:

```bash
export GUIDELINE_MD=su01-eval/unverifiable_bench/mo/rubrics/imo25_guideline.md
# or
export GUIDELINE_MD=su01-eval/unverifiable_bench/mo/rubrics/usamo2026_guideline.md
```
