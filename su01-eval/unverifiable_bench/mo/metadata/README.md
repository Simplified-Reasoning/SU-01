# MO Metadata

This folder stores problem metadata consumed by `prepare_mo_eval.py`.

- `imo25.jsonl`: IMO 2025 problem prompts, reference solutions, and metadata.
- `usamo2026.jsonl`: USAMO 2026 problem prompts, reference solutions, and
  metadata.

Example:

```bash
export META_JSONL=su01-eval/unverifiable_bench/mo/metadata/imo25.jsonl
# or
export META_JSONL=su01-eval/unverifiable_bench/mo/metadata/usamo2026.jsonl
```
